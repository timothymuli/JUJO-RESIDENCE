# Property photos

Drop your images here using the names below. The site uses `onerror` to show a grey placeholder until the file exists.

## JUJO Heights (Mlolongo) — folder `jujo-heights/`

| File | Suggested shot |
|------|----------------|
| `exterior-01.jpg` | Building front / gate |
| `exterior-02.jpg` | Parking or approach |
| `exterior-03.jpg` | Common area / garden |
| `interior-01.jpg` | Living area sample |
| `interior-02.jpg` | Kitchen sample |
| `interior-03.jpg` | Bedroom sample |
| `interior-04.jpg` | Bathroom sample |

Use **`.jpg`** or change the `src` in `mlolongo.html` to match (e.g. `.webp`). Keep files reasonably compressed for mobile.
